# 🎬 📚 culture-log | インプット＆感性の記録ノート

日々の読書、映画、ドラマの鑑賞ログを蓄積していくパーソナルリポジトリです。<br>
エンジニアとしての技術力だけでなく、**「日常のあらゆる作品から気付きを得るアンテナ」** と **「自分の言葉でアウトプットする言語化力」** を磨くために運用しています。

---

## 🎨 記録のジャンル

以下の3つのカテゴリーに分けて、それぞれの作品から得たインプットをMarkdown形式でまとめています。

### 📚 1. 読書ログ (`/books`)
技術書からビジネス書、小説、お気に入りの漫画まで幅広く記録しています。
単なるあらすじではなく、「実務やこれからの行動にどう活かすか」の視点を大切にしています。

### 📺 2. ドラマログ (`/dramas`)
国内外のドラマの鑑賞記録です。
連続したストーリーだからこそ描ける人間模様や、伏線回収の面白さを言葉に残しています。

### 🎬 3. 映画ログ (`/movies`)
作品のストーリー構成、映像美、心に刺さったセリフなどを記録しています。
クリエイティブな視点や、登場人物の心理描写を分析するのが好きです。
