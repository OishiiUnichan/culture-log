# 本当に腰が痛すぎて勘弁して欲しい。

一回書いた文章が全部消えてしまって悲しい。初めまして、うにです。  
ぎっくり腰がやばすぎて、本日は座って授業を受けるどころか動くのも辛いため職業訓練をお休みさせていただきました。  
整形外科のおじいちゃん「ヘルニア一歩手前だねえ」←そんなフランクに言わないでください  
接骨院のお兄さん「季節の変わり目には増えるんですよ」←流行りに乗ってるってことですか  
思えば毎年、この季節になるとぎっくり腰になっている、ような・・・そんな6月頭を過ごしています。  
ここまで前書きです。  

## 今日やったこと
整形外科でコルセットを入手し、多少楽になったのはいいんですが、家にいても暇なので授業のアウトプットを兼ねてGithubをいじり回すと決めて4時間。AIに助けてもらいながら色々できるんだなあとわかったところです。
- life-logを作る
- 勉強ログのアップロード（これは昨日だけど今日でいいや）
- 自己紹介のリポジトリを作る

特に最後のやつは学校では習ってないので自力で意外と辿り着けました。  
今日いっぱい使ったのはこのコマンドたち。流石に覚えたぞ。やはり手を動かすべき。  
```bash
git pull origin main
git add .
git commit -m "update"
git push origin main
```

今学校でやってるのはMySQLなので、SQLも手をガンガン動かしたいなとは思うが8GBのMacに仮想環境構築はちょっとな。  
ブラウザでいい感じにって思うとなかなか難しそうだなあと思っているところです。  

### ここから日記
普通にこれだけだと真面目な技術ログ風味になってしまうので日記を認めます。  
めっちゃ自語りなんですけど、私の実家はあんまり裕福ではないです。というか、平たくいうと貧乏育ちです。  
母は女で一つ、手取り12,3万円で3人の子供を大学や専門学校まで行かせてくれました。  
そんな母は自身が中卒で大変な思いをしてきたため、子供に学歴で不利な思いはさせまいと進学させてくれたというのを後々聞きました。  
今思うととんでもねえバイタリティーだなと思います、マネは絶対できないです。  
貧乏家庭にお小遣いなんてものは存在しなかったのですが、ど田舎で娯楽が全くないのでお金を必要とする場面もないせいかおかげか、母は「欲しいものはなんでも買い与える」というポリシーをもっていたそうです。  
その中でも、電子機器類はまじでお金に糸目をつけずに買ってくれました。そうです、ゲーム機とパソコンです。  
私には当時はデスクトップPCを、弟にはゲームを買ってくれてました。当時のデスクトップは16GBが安くても30万ぐらいした記憶があります。よくまあ、買ってくれたよな・・・と今になって思います。いや本当に。  


母いわく、「これからは絶対にパソコンがさわれたほうがいい」「ゲームでお金を稼げる時代は絶対来る」と思っていたそうです。  
先見の名がありすぎる。なんでそう思ってたの？と聞いた時に、「FFでファミコンのドット絵がポリゴンになった時にああいうのはパソコンで作ってるって知ったから。だから今後もっと発展したらターミネーターみたいな世界になると思ってる」そうです。と、とんでもねえ・・・
しかしながら、母は機械に強いわけではありません。むしろ「スマホの設定できないからやって〜泣」とスマホを持っていない当時の私に泣きつくほどの機械音痴です。  
インターネットの回線契約やwifiの初期セットアップなどは私が高校生になった頃にはすでに私に丸投げだった記憶があります。　　


紆余曲折はありましたが、私の人生は最終的にパソコンに関わるものになったし、ゲームが得意な弟はプロゲーマーへの誘いを何度も受けるほどになっています。弟たちは小学生の頃からFPSを嗜み、世界ランクの上位にいるのが当たり前みたいな人生だったからだろうなと思います。  


一方でお姉ちゃんの私は「特定の技術にこだわりがないのでなんでもやります」と振られた案件をこなしていたら「７割なんでもわかる使い勝手が悪くないエンジニア」になっていました。ただし、これが得意ですという領域もないです。  
個人的なポリシーとして「与えられた仕事はとりあえずやる」があるので、今は精神疾患とうまく付き合いながら続けられるものが見つかればいいなと思っています。  
得意な仕事は？って聞かれたらマニュアル整備で、AIが凄すぎるんでこの得意もきっといずれAIに奪われるんだろうなと思ったりもします。


そんなことを考えながら日記を認めてみました。いかがでしょうか。最後になりますが、ゼルダの伝説 風のタクトはGC世代のゼル伝の中で一番の名作だと思っているのでぜひみなさんもプレイしてみてくださいね。私の青春です。あとこのBGMは私がゲーム中ばかほど聞いた思い出です。神すぎる。
