## 今日やったこと
整形外科でコルセットを入手し、多少楽になったのはいいんですが、家にいても暇なので授業のアウトプットを兼ねてGithubをいじり回すと決めて4時間。AIに助けてもらいながら色々できるんだなあとわかったところです。
- life-logを作る
- 勉強ログのアップロード（これは昨日だけど今日でいいや）
- 自己紹介のリポジトリを作る

特に最後のやつは学校では習ってないので自力で意外と辿り着けました。  
今日いっぱい使ったのはこのコマンドたち。流石に覚えたぞ。やはり手を動かすべき。  
```bash
git pull origin main
git add .
git commit -m "update"
git push origin main
```

今学校でやってるのはMySQLなので、SQLも手をガンガン動かしたいなとは思うが8GBのMacに仮想環境構築はちょっとな。  
ブラウザでいい感じにって思うとなかなか難しそうだなあと思っているところです。
